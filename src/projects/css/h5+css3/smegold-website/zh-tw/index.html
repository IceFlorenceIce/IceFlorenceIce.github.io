自主研發的金融服務系統

我們專門為金融服務業客戶提供全面、整合的解決方案，以微小企業及中小企業為主的融資產品研究開發及市場行銷分析等領域

    • 應收賬款池融資軟件

      主要功能包含產品、公司/個人、征審、貸放款管理、收付款、貸款管理、風險管理、系統和報表。產品包含商品設計、商品管理、評分卡、審查條件管理、權重設置。

    • 風險預警系統

      為金融客戶提供基於大數據分析的壹站式風險控制與預警服務，降低信貸風險，提高信貸效率。

    • 小微貸款系統軟件

      主要功能包含產品、客戶管理、批發營銷、業務受理、貸款管理、風險管理、催收管理、業務管理、系統管理、報表、CIM客戶訊息管理分析、法金多維度分析。

    • 移動營銷管理系統

      主要功能包含客戶管理、業務受理、貸款管理、催收管理、系統工具、待辦事項。

    • 普惠信貸系統

      可快速、精準、高效的實現對個人、小微客戶的信貸產品營銷推廣、精準獲客，運用評分卡及貸後風險預警的模型，可提升客戶經理在管理風險上的效率。

    • 金融產品預審系統

      主要功能包含產品、客戶管理、案件管理、系統管理。產品包含產品管理、產品信息審核、題庫管理、產品篩選管理、產品利率審核、產品系列。

    • 線上快貸軟件

      主要功能包含申請端和管理端申請端包括貸款申請、我的貸款、管理入口等三個部分技術特點是快審快貸軟件采用混合架構。

    • 智能快貸系統

      用系統自動加人工輔助的方式進行進件處理、授信審核、用信放款，以達到快速營銷、精準獲客、智能審批的效果。

    • 積分計算軟件

      主要功能包含客戶管理、客戶積分管理、積分活動管理、兌換渠道管理、系統管理、日誌管理。

    • 智能辦公系統

      智能辦公管理系統大致規劃包括管理端和移動端，相對於管理端來說，移動端會更偏向於營銷人員、業務人員的應用，主要結合實際工作場景，進行功能設計和實現。

    • 人才測評軟件

      主要功能：包含求職者與員工測評、活動管理、企業管理、招聘管理、員工管理、導出表單、系統管理、收費管理。
