人力盤點

我們從客戶經理從事信貸工作年資分布、金融知 識筆試成績分布、專業能力培訓成績分布、產品 設計概念成績分布去分析和盤點。

業務流程優化及精細化管理

貸前主要提升貸前風險的管理績效，了解查訪進度、行程安排、問題處理及對申請客戶的資格審查。貸中著重在案件審批之後至撥款給申請客戶為止，依據規範準備審批資料。貸後明確制訂放款後的核實作業規範，核實貸款金額流向、貸後審查表、貸後征信審查、貸後財務報表等。

等級評定及考核機制設計

為了獎勵銷售進行業務拓展，並要求行政及銷售團隊對貸前調查、貸中審查及貸後風險進行有效管理，特設計了等級評定制度，通常將分為：定量考核、定性考核、貸後檢查、檔案管理四種，建立客戶經理淘汰與晉升機制，制訂每月貸款任務及客戶營銷數量，全面開展考核機制；設定小組管理模式，將客戶經理隊伍分組管理，並安排營銷組長擔任日常組內考核、貸款調查、輔導業務開展等工作；績效管理機制，建立日常崗位考核機制及季度考核淘汰機制相結合模式，實現績效管理公正、公開，發揮績效管理機制的導向性作用。

顧問式信貸培訓及輔助

透過培訓項目同時執行人力盤點計劃，針對全行客戶經理隊伍進行考察並據實評價，透過選拔後參加精英班培訓。對公對私客戶經理進行評級建議推選名單，為行方人力資源調度和業務管理提供合理依據。我們還參與每日支行的早晚會、協助營銷拜訪、現場調查並出具顧問建議書，以及時檢驗貸前調查技能，提高風險管控能力。加強內訓師隊伍建設，建立壹套完善的內訓師選拔機制，推進行內課程資源整合，優化培訓資源，不斷提升全體員工的職業素養和業務技能。

跨區經營及業務輔助

通過組織架構、產品市場、專業培訓、獎懲考核、風險管理、管理制度、獨立審核、科技平臺八個核心技術來完成貸款流程標準化，事業部績效管理，我們通過跨省分行、跨區經營、村鎮銀行三個經營方式來對銀行提供業務輔助。

智能快貸系統

智能快貸項目基本需求為：結合銀行內外部數據，利用大數據理念，針對不同市民金融產品設計和建立不同的模型，只需客戶簡單錄入，即可給出信用評級和預授信額度，同時銀行通過後臺能更快速收集和核查該客戶相關重要信息，用系統自動加人工輔助的方式進行進件處理、授信審核、用信放款， 以達到快速營銷、精準獲客、智能審批的效果。該平臺包含公共客戶端（H5、公眾號、小程序）、銀行員工端（APP）和銀行管理端（WEB/PC）三個子平臺，其中公共客戶端主要提供給公眾客戶使用，做貸款申請和貸款查詢使用；銀行員工端主要是行內客戶經理、審查崗、貸後崗、信息采集崗等進行業務移動辦公；銀行管理端主要是行內管理崗、客戶經理、平臺運營人員使用，做模型配置、系統管理、業務管理、日常管理等。
