公司簡介

Company Profile

中小金流資訊股份有限公司，總部設在臺灣，內地中小融輔軟件科技（上海）有限公司為中小金流資訊股份有限公司在內地的分公司。核心技術發源於美國融資財務公司，引入先進的財務管理方式，致力於為亞洲地區政府單位以及金融機構，提供有關中小微企業融資顧問服務，協助金融機構拓展中小微企業融資業務，強化風險管理，並使中小微企業順利取得運營資金，促進經濟及產業繁榮。

成立於2005年的中小金流／中小融輔公司總部及產品研發團隊設於臺灣臺北，系統技術研發基地位於中國大陸上海，營銷團隊設於中國大陸上海及浙江杭州。服務面向包括「顧問輔導→系統平臺→專業培訓→數據挖掘→服務創新」的壹條龍服務，並由零售金融為基礎，逐步帶入融資、保險、理財及信用卡的金融服務項目，開啟第三方金融服務平臺與銀行間的合作通道。
