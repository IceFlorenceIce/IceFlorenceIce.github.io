南京銀行成立於1996年2月8日，是壹家具有由國有股份、中資法人股份、外資股份及眾多個人股份共同組成獨立法人資格的股份制商業銀行，實行壹級法人體制。南京銀行歷經兩次更名，先後於2001年、2005年引入國際金融公司和法國巴黎銀行入股，在全國城商行中率先啟動上市輔導程序並於2007年成功上市。

註冊資本：人民幣29.69億元

分支行網點：9 家分行（泰州、北京、上海、杭州、揚州、無錫、南通、蘇州、常州）

東北地區

長春發展農商銀行

長春發展農商銀行全稱長春發展農村商業銀行股份有限公司，前身是長春市環城農村信用合作聯社，始建於1988年。總部位於長春市經濟技術開發區，註冊資本10.075億元，共有員工1035人，內設職能部門17個，下轄支行58家，其中城區支行37家、農村支行21家，服務範圍覆蓋長春市區和周邊鄉鎮。

臺港澳地區

臺灣經濟部中小企業處（27家銀行）

前系『臺灣無盡株式會社 』於民國四年六月(西元壹九壹五年)創立，總公司設在臺北市。民國三十五年九月壹日悉由臺灣省行政長官公署將『臺灣勸業無盡』、『臺灣南部無盡』、『東臺灣無盡』、『臺灣住宅無盡』等四家株式會社合併改組為『臺灣無盡業股份有限公司』。民國三十六年六月改稱『臺灣省人民貯金互濟股份有限公司』。三十七年壹月更名為『臺灣合會儲蓄股份有限公司』。民國六十五年七月壹日改制為『臺灣中小企業銀行股份有限公司』，為我國首先創設成立，以提供中小企業融資與輔導為宗旨，使本行真正由當年的合會儲蓄機構，蛻變成擁有完整金融服務功能的專業銀行。民國八十七年壹月二十二日改制為民營銀行。
