自主研发的金融服务系统

我们专门为金融服务业客户提供全面、整合的解决方案，以微小企业及中小企业为主的融资产品研究开发及市場行銷分析等领域

    • 应收账款池融资软件

      主要功能包含产品、公司/个人、征审、贷放款管理、收付款、贷款管理、风险管理、系统和报表。产品包含商品设计、商品管理、评分卡、审查条件管理、权重设置。

    • 风险预警系统

      为金融客户提供基于大数据分析的一站式风险控制与预警服务，降低信贷风险，提高信贷效率。

    • 小微贷款系统软件

      主要功能包含产品、客户管理、批发营销、业务受理、贷款管理、风险管理、催收管理、业务管理、系统管理、报表、CIM客户讯息管理分析、法金多维度分析。

    • 移动营销管理系统

      主要功能包含客户管理、业务受理、贷款管理、催收管理、系统工具、待办事项。

    • 普惠信贷系统

      可快速、精准、高效的实现对个人、小微客户的信贷产品营销推广、精准获客，运用评分卡及贷后风险预警的模型，可提升客户经理在管理风险上的效率。

    • 金融产品预审系统

      主要功能包含产品、客户管理、案件管理、系统管理。产品包含产品管理、产品信息审核、题库管理、产品筛选管理、产品利率审核、产品系列。

    • 线上快贷软件

      主要功能包含申请端和管理端申请端包括贷款申请、我的贷款、管理入口等三个部分技术特点是快审快贷软件采用混合架构。

    • 智能快贷系统

      用系统自动加人工辅助的方式进行进件处理、授信审核、用信放款，以达到快速营销、精准获客、智能审批的效果。

    • 积分计算软件

      主要功能包含客户管理、客户积分管理、积分活动管理、兑换渠道管理、系统管理、日志管理。

    • 智能办公系统

      智能办公管理系统大致规划包括管理端和移动端，相对于管理端来说，移动端会更偏向于营销人员、业务人员的应用，主要结合实际工作场景，进行功能设计和实现。

    • 人才测评软件

      主要功能：包含求职者与员工测评、活动管理、企业管理、招聘管理、员工管理、导出表单、系统管理、收费管理。
